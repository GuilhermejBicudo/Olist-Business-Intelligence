# Olist Business Intelligence

End-to-End Data Analytics project using Python, PostgreSQL and Power BI.

---

## Project Overview

This project simulates a real Business Intelligence initiative for a Brazilian e-commerce company.

The objective is to transform raw transactional data into strategic information through data cleaning, modeling, SQL analysis, and interactive dashboards.

Instead of focusing only on visualization, this project follows the complete analytics lifecycle used in real companies.

---

## Business Problem

Suppose you were hired as a Data Analyst by Olist.

Management wants to answer questions such as:

- Which customers generate the highest revenue?
- Which product categories are the most profitable?
- How are sales evolving over time?
- Which sellers have the best performance?
- Which regions represent the biggest opportunities?
- How efficient is the delivery process?
- What factors influence customer satisfaction?
